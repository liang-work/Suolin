# 如何启动该项目

## 桌面端（Windows/Linux/macOS）

前往[ Python 官网](https://python.org/)下载您对应平台的 Python（版本应 >=3.11.x）。随后，安装必须的库。

## 移动端（Android）

前往[ Termux 官网](https://termux.dev/)下载并安装。随后运行

```shell
pkg update&&pkg install python
```

完成后安装必要的库，运行本项目。


## 移动端（iOS/iPadOS）

前往应用商店下载[‎a-Shell App - App Store](https://apps.apple.com/us/app/a-shell/id1473805438)，下载完成后安装必须的库，然后运行本项目。
